# Smart Crop Advisory System

This is a code bundle for Smart Crop Advisory System. The original project is available at https://www.figma.com/design/R4Ho8DRoYqvb1jEqE1rWC6/Smart-Crop-Advisory-System.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Mini search engine in C

A small example search engine is included in `mini_search_engine.c`.

### Compile

```bash
gcc -O2 -std=c11 mini_search_engine.c -o mini_search
```

### Run

```bash
./mini_search file1.txt file2.txt file3.txt
```

Then type queries at `search>` and type `exit` to quit.
